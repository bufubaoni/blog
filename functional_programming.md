# 函数式编程
实际编程过程中，总在追求可读性，很自然的就将函数写成了函数式。
当然我所讲的函数式并非数学上的函数式，仅仅希望将可读性增强一点。

最初理解的函数式，就仅仅是个函数而已。没有那么多oop的属性。这有什么美观可言呢。

```python
def fn(x):
    return x + 1 
```
一个函数的返回值仅仅依靠他的输入值。

这样我们就可以将几个函数串联起来

``` py
def fn1(x):
    return x * x

fn1(fn(x))
```
我们就可以使用这样的代码来调用。

如果可读性不方便 那么 可以这样利用高级特性来调用
```py
reduce(x,[fn, fn1])
```
这样我们可以任意的添加几个计算规则，并且不会影响到输出
至此可能就是我理解的可读性了。
